## setup

install nodemon for hot reload

```
npm install nodemon -g
```

install packages

```
npm install
```

start server using nodemo

```
npm start
```

start server without nodemon(no hot reloading)

```
node server.js
```

generate json data

```
node factory/<js-file-name>
// ~~.jsonを作成しましたと出ればおk
```
